# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by joining the [Discord Server](https://discord.gg/xxQmAhzxvP).

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact (if known)

---

## Support Policy

- We will assist with vulnerabilities found in the **original, unmodified codebase**
- If you have modified the code in any way, **support will not be provided**
- Issues caused by custom changes or self-hosted setups are your responsibility

---

## Response Time

We aim to:
- Acknowledge reports within 48 hours
- Investigate and resolve valid issues as soon as possible

---

## Disclaimer

This project is provided "as is" without warranty.  
Security is important, but no system is guaranteed to be completely secure.
